# 08. Personal Details (Screen 3)

This API is used to save the user's Lifestyle and Physical attributes.

**URL:**
`/api/auth/user/user-personal-details`

**Method:**
Post

**Headers:**
```
Authorization: Bearer {{token}}
x-app-type: user
Content-Type: application/json
Accept-Language: en-US
```

**Sample Input**

```json
{
    "diet": "Veg",
    "smoke": "No",
    "drink": "Occasionally",
    "bodyType": "Slim",
    "skinTone": "Fair",
    "anyDisability": "No",
    "heightFeet": 5,
    "heightInches": 11,
    "userWeight": 70
}
```

**Sample Output (Success)**

```json
{
    "message": "Personal details saved successfully.",
    "statusCode": 200,
    "messageCode": "success.profilePersonalSaved",
    "profileCompleteness": {
        "id": "00ddbfc79419419ab3ad63b2df24a774",
        "userId": "b67695843eaa4f6b9caa1c2ad105006a",
        "createdAt": 1779117726,
        "updatedAt": 1779118818,
        "userDetails": true,
        "userBasicDetails": true,
        "professionDetails": true,
        "personalDetails": true,
        "galleryDetails": false,
        "familyDetails": false,
        "addressDetails": false,
        "partnerPreferencesDetails": false,
        "verifyMobileNumber": true
    }
}
```

**Error Message Scenarios**

1. Token Required / Invalid
```json
{
  "message": "Authorization token is required.",
  "statusCode": 401,
  "messageCode": "error.tokenRequired"
}

```

2. Missing Fields
```json
{
  "message": "userWeight is required.",
  "statusCode": 400,
  "messageCode": "error.fieldRequired"
}

```
