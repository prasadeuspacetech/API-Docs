# 01. User Signup (Send OTP)

This API is used to register a new user and send an OTP to their mobile number.

**URL:**
`/api/user/signup`

**Method:**
Post

**Headers:**
```
Content-Type: application/json
Accept-Language: en-US
```

**Sample Input**

```json
{
    "mobileNumber": "9000000125",
    "countryCode": "+91",
    "firstName": "Dummy.Nitin",
    "lastName": "Mehta",
    "email": "nitin.mehta12@example.com",
    "gender": "Male",
    "birthDate": "15-06-2004",
    "deviceInfo": {
        "deviceId": "devM05",
        "deviceType": "android",
        "os": "android",
        "osVersion": "14",
        "appVersion": "1.0",
        "notificationToken": "token-M05"
    }
}
```

**Error Message Scenarios**

1. Mobile Number Already Registered
```json
{
    "message": "Mobile number already registered. Please login.",
    "statusCode": 400,
    "messageCode": "error.mobileAlreadyRegistered"
}
```

2. Email Address Already Registered
```json
{
    "message": "Email address already registered. Please use another email.",
    "statusCode": 400,
    "messageCode": "error.emailAlreadyRegistered"
}
```

3. Missing Required Fields
```json
{
    "message": "mobileNumber and countryCode is required.",
    "statusCode": 400,
    "messageCode": "error.fieldRequired"
}
```

4. OTP Rate Limit (Too many requests)
```json
{
  "message": "Too many OTP requests. Please try again after 10 minutes.",
  "statusCode": 429,
  "messageCode": "error.otpRateLimit"
}
```

5. Invalid JSON Body
```json
{
    "message": "Invalid JSON body.",
    "statusCode": 400,
    "messageCode": "error.invalidJson"
}
```
