# 01. User Signup (Send OTP)

This API is used to register a new user and send an OTP to their mobile number.

**URL:**
`/api/user/signup`

**Method:**
Post

**Headers:**
```
Content-Type: application/json
```

**Sample Input**

```json
{
    "mobileNumber": "1234567885",
    "countryCode": "+91",

    "lastName": "Test B",

    "birthDate": "17-12-1996",
    "deviceInfo": {
        "deviceId": "sada",
        "deviceType": "android",
        "os": "android",
        "osVersion": "1.2",
        "appVersion": "1.0",
        "notificationToken": "token-123..."
    }
}
```

**Error Message Scenarios**

1. Mobile Number Already Registered
```json
{
    "message": "Mobile number already registered. Please login."
}
```

2. Email Address Already Registered
```json
{
    "message": "Email address already registered. Please use another email."
}
```

3. Missing Required Fields
```json
{
    "message": "firstName is required."
}
```

4. OTP Rate Limit (Too many requests)
```json
{
    "message": "Too many OTP requests. Please try again after 10 minutes."
}
```

5. Invalid JSON Body
```json
{
    "message": "Invalid JSON body."
}
```
