# 11. Premium Matches

This API fetches high-quality matches based on a shared cultural background (Religion, Caste, Subcaste) and location (Country, State, City), with specific age-gender logic.

**URL:**
`/api/auth/user/dashboard/premium`

**Method:**
`GET`

**Headers:**
```
Authorization: Bearer {{token}}
x-app-type: user
```

**Logic Detail:**
*   **Background Match**: Profiles must match the user's **Partner Preferences** for Religion, Caste, Subcaste, Country, State, and City.
*   **Gender**: Opposite gender only.
*   **Age Logic**:
    *   **Male users**: See females aged **[User Age - 4]** to **[User Age]**.
    *   **Female users**: See males aged **[User Age + 1]** to **[User Age + 5]**.

**Query Parameters:**

| Parameter | Type | Default | Description |
| :--- | :--- | :--- | :--- |
| `limit` | `integer` | `5` | The number of items to return. Use `3` for Recommendations and `5` for Matches tab. |
| `page` | `integer` | `1` | The page number to fetch. |
