# 04. Verify Login OTP

This API is used to verify the OTP for an existing user. On successful verification, the user is authenticated and receives their session token and profile data.

**URL:**
`/api/user/verify-login-otp`

**Method:**
Post

**Headers:**
```
Content-Type: application/json
```

**Sample Input**

```json
{
    "mobileNumber": "1234567885",
    "countryCode": "+91",
    "otp": "123456",
    "deviceInfo": {
        "deviceId": "sada",
        "deviceType": "android",
        "os": "android",
        "osVersion": "1.2",
        "appVersion": "1.0",
        "notificationToken": "token-123..."
    }
}
```

**Error Message Scenarios**

1. Invalid OTP
```json
{
    "message": "Invalid OTP. Please try again."
}
```

2. OTP Expired
```json
{
    "message": "OTP has expired. Please request a new one."
}
```

3. User Not Found
```json
{
    "message": "User not found."
}
```
