# 04. Verify Login OTP

This API is used to verify the OTP for an existing user. On successful verification, the user is authenticated and receives their session token and profile data.

**URL:**
`/api/user/verify-login-otp`

**Method:**
Post

**Headers:**
```
Content-Type: application/json
Accept-Language: hi
```

**Sample Input**

```json
{
  "mobileNumber": "9000000125",
  "countryCode": "+91",
  "otp": "4866",
  "deviceInfo": {
    "deviceId": "device-001",
    "deviceType": "android",
    "os": "Android",
    "osVersion": "13",
    "appVersion": "1.0.0",
    "notificationToken": "fcm-token-001"
  }
}
```

**Error Message Scenarios**

1. Invalid OTP
```json
{
  "message": "Invalid OTP. Please try again.",
  "statusCode": 400,
  "messageCode": "error.otpInvalid"
}

```

2. OTP Expired
```json
{
  "message": "OTP has expired. Please request a new one.",
  "statusCode": 400,
  "messageCode": "error.otpExpired"
}

```

3. User Not Found
```json
{
  "message": "User not found.",
  "statusCode": 400,
  "messageCode": "error.userNotFound"
}

```
