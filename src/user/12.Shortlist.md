# 12. User Shortlist

This API allows users to manage a personalized list of profiles they are interested in.

**URL:**
`/api/auth/user/shortlist`

**Headers:**
```
Authorization: Bearer {{token}}
x-app-type: user
Content-Type: application/json
Accept-Language: en-US
```

---

## 1. Add to Shortlist
Adds a user to your shortlist.

**Method:**
`POST`

**Sample Input:**
```json
{
    "target_user_id": "2cbe1b59e8c44600986c3aa5979a0c06"
}
```

**Sample Output (Success):**
```json
{
    "message": "User shortlisted successfully",
    "statusCode": 200,
    "messageCode": "success.shortlistAdded"
}
```

---

## 3. Remove from Shortlist
Removes a user from your shortlist (Soft Delete).

**Method:**
`DELETE`

**Sample Input:**
```json
{
    "target_user_id": "2cbe1b59e8c44600986c3aa5979a0c06"
}
```

**Sample Output (Success):**
```json
{
  "message": "User removed from shortlist",
  "statusCode": 200,
  "messageCode": "success.shortlistRemoved"
}

```
