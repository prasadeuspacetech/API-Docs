# 14. Get Profile By ID

This API fetches the full consolidated profile of a user using their internal UUID.

**URL:**
`/api/auth/profile/details-id/{{user_id}}`

**Method:**
`GET`

**Headers:**
```
Authorization: Bearer {{token}}
x-app-type: user
```

**URL Parameters:**
| Parameter | Type | Description |
| :--- | :--- | :--- |
| `user_id` | `string` | The internal unique identifier (UUID) of the user. |
