# Simple Testing Guide for Chat APIs

## 1. Important Note Before Testing
Users can ONLY chat if they are connected as "Friends" in the system. Make sure the two users you are testing with have an accepted friend request.

## 2. Testing the Real-Time Chat (WebSocket)

WebSockets (`ws://`) are used for the live chat connection so messages appear instantly.


**How to test in Postman:**
1. Open Postman.
2. Click **New** -> **WebSocket Request**.
3. Enter this URL for User A:
   `	ext
   ws://bandhan-setu-prod.eba-am6hwsad.ap-south-1.elasticbeanstalk.com/api/auth/chat
   `
4. Go to the **Headers** tab in Postman and add:
   * Key: Authorization, Value: Bearer PASTE_USER_A_TOKEN_HERE
5. Click **Connect**.
6. Open a second Postman WebSocket tab for User B and connect:
   `	ext
   ws://bandhan-setu-prod.eba-am6hwsad.ap-south-1.elasticbeanstalk.com/api/auth/chat
   `
7. Go to the **Headers** tab for User B and add:
   * Key: Authorization, Value: Bearer PASTE_USER_B_TOKEN_HERE
8. Click **Connect**.

**To send a message:**
In User A's Postman tab, select the "Message" box at the bottom, make sure it's set to JSON, paste this, and click "Send":
```json
{
  "action": "send_message",
  "receiver_id": "PASTE_USER_B_ID_HERE",
  "message_text": "Hi! This is a test message."
}
```
User B will instantly see the message appear in their Postman tab!

## 3. Testing the Standard APIs (HTTP)

These are normal REST APIs. You test them like any other API in Postman (using `http://`).

**Required Headers for all APIs below:**
* `Authorization`: `Bearer PASTE_TOKEN_HERE`
* `x-app-type`: `user`

---

### A. Get Chat Inbox (List of Conversations)
* **Method:** `GET`
* **URL:** `http://bandhan-setu-prod.eba-am6hwsad.ap-south-1.elasticbeanstalk.com/api/auth/chat/inbox?page=1&limit=20`
* **What it does:** Shows all users you've chatted with and your latest messages.
* **Successful Response (200 OK - Raw JSON Array):**
  ```json
  [
      {
          "id": "user_id_hex",
          "platformId": "BS12345",
          "firstName": "John",
          "lastName": "Doe",
          "gender": "Male",
          "profilePicture": "photo_hash.jpg",
          "lastMessageId": "msg_id_hex",
          "lastMessageSenderId": "user_id_hex",
          "lastMessageText": "Hi! This is a test message.",
          "lastMessageStatus": "delivered",
          "lastMessageTime": 1716180000,
          "lastMessageDeleted": 0,
          "unreadCount": 2
      }
  ]
  ```

### B. Get Chat History (Messages with a specific user)
* **Method:** `GET`
* **URL:** `http://bandhan-setu-prod.eba-am6hwsad.ap-south-1.elasticbeanstalk.com/api/auth/chat/history/PASTE_OTHER_USER_ID_HERE?page=1&limit=20`
* **What it does:** Shows all the past messages between you and that user.
* **Successful Response (200 OK - Raw JSON Array):**
  ```json
  [
      {
          "id": "msg_id_hex",
          "senderId": "sender_user_id_hex",
          "receiverId": "receiver_user_id_hex",
          "messageText": "Hi! This is a test message.",
          "messageStatus": "read",
          "createdAt": 1716180000,
          "deleted": false
      },
      {
          "id": "msg_id_hex_2",
          "senderId": "receiver_user_id_hex",
          "receiverId": "sender_user_id_hex",
          "messageText": "Hello! How are you?",
          "messageStatus": "read",
          "createdAt": 1716179000,
          "deleted": false
      }
  ]
  ```

### C. Delete a Message
* **Method:** `DELETE`
* **URL:** `http://bandhan-setu-prod.eba-am6hwsad.ap-south-1.elasticbeanstalk.com/api/auth/chat/message/PASTE_MESSAGE_ID_HERE`
* **What it does:** Deletes a specific message. (You must be the sender to delete it).
* **Successful Response (200 OK - Raw JSON Object):**
  ```json
  {
      "success": true
  }
  ```
