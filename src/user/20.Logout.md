# Logout API

Invalidates the user's current session token. Once this API is called, the JWT token provided in the authorization header will no longer be valid for any authenticated endpoints.

**Endpoint:** `POST /api/auth/user/logout`

**Authentication:** Required (Bearer Token)

### Headers

| Key | Value | Description |
| :--- | :--- | :--- |
| `Authorization` | `Bearer <your_jwt_token>` | **Required**. The valid JWT token. |
| `x-app-type` | `user` | **Required**. Identifies the client platform. |

### Request Body

None required.

### Success Response

**Status Code:** 200 OK

```json
{
    "message": "Logged out successfully."
}
```

### Error Responses

**401 Unauthorized**
This occurs if the token is already expired, missing, or previously logged out.

```json
{
    "message": "Unauthorized or session expired",
    "error_code": 2002
}
```
