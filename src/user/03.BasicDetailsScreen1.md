# 06. Basic Details (Screen 1)

This API is used to save the user's Religion and Location details.

**URL:**
`/api/auth/user/user-basic-details`

**Method:**
Post

**Headers:**
```
Authorization: Bearer {{token}}
Content-Type: application/json
Accept-Language: en-US
```

**Sample Input**

```json
{
    "religionName": "Hindu",
    "castName": "Brahmin",
    "subcastName": "Saraswat",
    "countryName": "India",
    "stateName": "Maharashtra",
    "cityName": "Pune",
    "maritalStatus": "Never Married",
    "childStatus": "No",
    "numberOfChildrens": 0,
    "mothertongueName": "Marathi"
}
```

**Error Message Scenarios**

1. Token Required / Invalid
```json
{
  "message": "Authorization token is required.",
  "statusCode": 401,
  "messageCode": "error.tokenRequired"
}

```

2. Missing Fields
```json
{
  "message": "religionName is required.",
  "statusCode": 400,
  "messageCode": "error.fieldRequired"
}

```

3. Database Error
```json
{
  "message": "Database error occurred",
  "statusCode": 400,
  "messageCode": "error.dbError"
}

```
