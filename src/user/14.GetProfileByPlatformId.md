# 15. Get Profile By Platform ID

This API fetches the full consolidated profile of a user using their human-readable Platform ID.

**URL:**
`/api/auth/profile/details-platform/{{platform_id}}`

**Method:**
`GET`

**Headers:**
```
Authorization: Bearer {{token}}
x-app-type: user
Accept-Language: en-US
```

**URL Parameters:**
| Parameter | Type | Description |
| :--- | :--- | :--- |
| `platform_id` | `string` | The public platform identifier (e.g., M-123667). |
