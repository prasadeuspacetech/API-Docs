# 10. Compatibility Matches

This API is used to fetch a list of potential partner matches based on the logged-in user's age, gender, and saved partner preferences. This serves as the "Compatibility" sub-tab in the dashboard.

**URL:**
`/api/auth/user/dashboard/compatibility`

**Method:**
`GET`

**Headers:**
```
Authorization: Bearer {{token}}
x-app-type: user
```

**Query Parameters:**

| Parameter | Type | Default | Description |
| :--- | :--- | :--- | :--- |
| `limit` | `integer` | `5` | The number of items to return. Use `3` for Recommendations and `5` for Matches tab. |
| `page` | `integer` | `1` | The page number to fetch. |
