# 12. New Matches

This API fetches profiles of users who share the same background and exact same age, and who have joined the platform within the last 14 days.

**URL:**
`/api/auth/user/dashboard/new-matches`

**Method:**
`GET`

**Headers:**
```
Authorization: Bearer {{token}}
x-app-type: user
```

**Logic Detail:**
*   **Background Match**: Profiles must match the user's **Partner Preferences** for Religion, Caste, Subcaste, Country, State, and City.
*   **Age Match**: Profiles must be the **exact same age** as the user.
*   **Recency**: Profiles must have joined within the **last 14 days**.

**Query Parameters:**

| Parameter | Type | Default | Description |
| :--- | :--- | :--- | :--- |
| `limit` | `integer` | `5` | The number of items to return. Use `3` for Recommendations and `5` for Matches tab. |
| `page` | `integer` | `1` | The page number to fetch. |
