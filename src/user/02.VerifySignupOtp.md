# 02. Verify Signup OTP

This API is used to verify the OTP sent to the user's mobile number during registration. On successful verification, the user is logged in and a session token is provided.

**URL:**
`/api/auth/user/verify-otp`

**Method:**
Post

**Headers:**
```
Authorization: Bearer {{token}}
x-app-type: user
Content-Type: application/json
```

**Sample Input**

```json
{
    "mobileNumber": "1234567890",
    "countryCode": "+91",
    "otp": "123456"
}
```

**Sample Output (Success)**

```json
{
    "message": "Registration successful.",
    "profileCompleteness": {
        "id": "5e26daa1b0de4d69996367953db53c50",
        "createdAt": 1776803572000,
        "updatedAt": 1776803572000,
        "userBasicDetails": true,
        "verifyMobileNumber": true,
        "userDetails": true,
        "professionDetails": false,
        "personalDetails": false,
        "galleryDetails": false,
        "familyDetails": false,
        "addressDetails": false,
        "partnerPreferencesDetails": false,
        "userId": "b3bcfb46eb8340caae831878ed7f84d8"
    }
}
```

**Error Message Scenarios**

1. Invalid OTP
```json
{
    "message": "Invalid OTP. Please try again."
}
```

2. OTP Expired
```json
{
    "message": "OTP has expired. Please request a new one."
}
```

3. OTP Not Found (Never requested)
```json
{
    "message": "No OTP found. Please request a new one."
}
```

4. Maximum Attempts Exceeded
```json
{
    "message": "Maximum OTP attempts exceeded. Please request a new one."
}
```

5. User Not Found
```json
{
    "message": "User not found."
}
```
