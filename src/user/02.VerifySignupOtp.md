# 02. Verify Signup OTP

This API is used to verify the OTP sent to the user's mobile number during registration. On successful verification, the user is logged in and a session token is provided.

**URL:**
`/api/auth/user/verify-otp`

**Method:**
Post

**Headers:**
```
Authorization: Bearer {{token}}
x-app-type: user
Content-Type: application/json
Accept-Language: en-US
```

**Sample Input**

```json
{
  "mobileNumber": "9000000125",
  "countryCode": "+91",
  "otp": "1453"
}
```

**Sample Output (Success)**

```json
{
    "statusCode": 200,
    "messageCode": "success.otpVerified",
    "message": "OTP verified successfully.",
    "profileCompleteness": {
        "id": "00ddbfc79419419ab3ad63b2df24a774",
        "userId": "b67695843eaa4f6b9caa1c2ad105006a",
        "createdAt": 1779117726,
        "updatedAt": 1779117808,
        "userDetails": true,
        "userBasicDetails": false,
        "professionDetails": false,
        "personalDetails": false,
        "galleryDetails": false,
        "familyDetails": false,
        "addressDetails": false,
        "partnerPreferencesDetails": false,
        "verifyMobileNumber": true
    }
}
```

**Error Message Scenarios**

1. Invalid OTP
```json
{
  "message": "Invalid OTP. Please try again.",
  "statusCode": 400,
  "messageCode": "error.otpInvalid"
}

```

2. OTP Expired
```json
{
  "message": "OTP has expired. Please request a new one.",
  "statusCode": 400,
  "messageCode": "error.otpExpired"
}

```

3. OTP Not Found (Never requested)
```json
{
    "message": "No OTP found. Please request a new one.",
    "statusCode": 400,
    "messageCode": "error.otpNotFound"
}
```

4. Maximum Attempts Exceeded
```json
{
  "message": "Maximum OTP attempts exceeded. Please request a new one.",
  "statusCode": 400,
  "messageCode": "error.otpMaxAttempts"
}

```

5. User Not Found
```json
{
  "message": "User not found.",
  "statusCode": 400,
  "messageCode": "error.userNotFound"
}

```
