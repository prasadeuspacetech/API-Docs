# 07. Professional Details (Screen 2)

This API is used to save the user's Education and Career information.

**URL:**
`/api/auth/user/user-professional-details`

**Method:**
Post

**Headers:**
```
Authorization: Bearer {{token}}
x-app-type: user
Content-Type: application/json
```

**Sample Input**

```json
{
    "collegeName": "IIT Bombay",
    "educationLevelName": "Bachelor's",
    "educationFieldName": "Computer Science",
    "employerName": "Google",
    "annualIncomeName": "20L - 30L",
    "currencyType": "INR",
    "workingWithName": "Private Sector",
    "categoryName": "Software Professional",
    "subcategoryName": "Backend Developer"
}
```


**Error Message Scenarios**

1. Token Required / Invalid
```json
{
    "message": "Authorization token is required."
}
```

2. Missing Fields
```json
{
    "message": "educationLevelName is required."
}
```

3. Database Error
```json
{
    "message": "Failed to save profile details. Please try again.",
    "data": {}
}
```
