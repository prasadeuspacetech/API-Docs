# Get Self Profile

This API fetches the full consolidated profile of the logged-in user. It does not require any parameters in the URL because it identifies the user automatically from the Authorization Token.

**URL:**
`/api/auth/user/my-profile`

**Method:**
`GET`

**Headers:**
```
Authorization: Bearer {{token}}
x-app-type: user
accept-language : en or hi
```

**URL Parameters:**
None
