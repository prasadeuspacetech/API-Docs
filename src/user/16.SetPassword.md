# 06. Set Password

This API allows an authenticated user to set or update their account password.

**URL:**
`/api/auth/user/set-password`

**Method:**
`POST`

**Headers:**
```
Content-Type: application/json
Authorization: Bearer <your_jwt_token>
Accept-Language: en-US
```

**Sample Input**

```json
{
  "password": "man@123"
}
```

**Sample Output (Success)**

```json
{
    "message": "Password set successfully.",
    "statusCode": 200,
    "messageCode": "success.passwordSet"
}
```

**Error Message Scenarios**

1. **Password Too Short** (Minimum 6 characters)
```json
{
  "message": "Password must be at least 6 characters.",
  "statusCode": 400,
  "messageCode": "error.passwordMinLength"
}

```

2. **Missing Password Field**
```json
{
  "message": "password is required.",
  "statusCode": 400,
  "messageCode": "error.fieldRequired"
}

```

3. **Invalid or Missing Token**
```json
{
  "message": "Authorization token is required.",
  "statusCode": 401,
  "messageCode": "error.tokenRequired"
}

```
OR
```json
{
  "message": "Invalid or expired token.",
  "statusCode": 401,
  "messageCode": "error.invalidToken"
}

```

4. **Invalid JSON Body**
```json
{
  "message": "Invalid JSON body.",
  "statusCode": 400,
  "messageCode": "error.invalidJson"
}

```
