# Admin API: Get User Full Profile

This API is used by the admin panel to fetch the complete, consolidated profile of any user in the system using their unique `user_id`.

## **Endpoint Details**
* **URL:** `/api/auth/admin/userprofile/{{user_id}}`
* **Method:** `GET`
* **Headers:**
  * `Authorization`: `Bearer <admin_jwt_token>`
  * `x-app-type`: `admin`

## **Description**
Returns the full, logically grouped profile data for the requested user exactly as it is structured for the frontend. This includes their basic details, primary details, education, working, lifestyle, about me, address, family, and their overall profile completeness metrics. 

*(Note: Friend request and shortlist logic checks are bypassed since the admin is viewing the profile globally.)*

## **Error Responses**

### **404 Not Found**
Returned if the `user_id` does not exist in the database or has been permanently deleted.
```json
{
    "status": "error",
    "message": "User not found.",
    "data": null
}
```

### **401 Unauthorized**
Returned if the admin JWT token is missing, invalid, or expired.
```json
{
    "status": "error",
    "message": "Invalid or expired token.",
    "data": null
}
```

### **403 Forbidden**
Returned if the request is made by a standard user instead of an admin (`x-app-type: user`).
```json
{
    "status": "error",
    "message": "Admin app type header required.",
    "data": null
}
```
