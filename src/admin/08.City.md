# 07. City CRUD

This API is used by Admins to manage cities.

**Base URL:** `/api/auth/admin/city`

---

## 1. List Cities
Returns a list of cities. Can be filtered by `state_id`.

**URL:** `/api/auth/admin/city`
**Method:** `GET`
**Headers:**
`Authorization: Bearer <Admin_Token>`
`x-app-type: admin`
**Query Parameters:**
| Parameter | Type | Required | Description |
| :--- | :--- | :--- | :--- |
| `state_id` | String | No | Filter cities by state |

**Response:**
```json
{
    "cities": [
        {
            "id": "ct001",
            "city_name": "Mumbai",
            "state_id": "st001"
        }
    ],
    "message": "Success"
}
```

---

## 2. Add City
Adds a new city.

**URL:** `/api/auth/admin/city`
**Method:** `POST`
**Headers:**
`Authorization: Bearer <Admin_Token>`
`x-app-type: admin`

**Request Body:**
```json
{
    "city_name": "Pune",
    "state_id": "st001"
}
```

**Response:**
```json
{
    "message": "City added successfully"
}
```

---

## 3. Update City
Updates an existing city name.

**URL:** `/api/auth/admin/city/<city_id>`
**Method:** `PUT`
**Headers:** `Authorization: Bearer <Admin_Token>`

**Request Body:**
```json
{
    "city_name": "Pune Updated"
}
```

**Response:**
```json
{
    "message": "City updated successfully"
}
```

---

## 4. Delete City
Soft deletes a city.

**URL:** `/api/auth/admin/city/<city_id>`
**Method:** `DELETE`
**Headers:** `Authorization: Bearer <Admin_Token>`

**Response:**
```json
{
    "message": "City deleted successfully"
}
```
