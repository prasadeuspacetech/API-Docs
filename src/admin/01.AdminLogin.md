# 01. Admin Login

This API is used to authenticate an admin and receive a session token.

**URL:**
`/admin/auth/login`

**Method:**
`POST`

**Headers:**
```
Accept-Language: en-US
Content-Type: application/json
```

**Sample Input**

```json
{
    "email":"admin@bandhansetu.com",
    "password":"Admin@123",
    "device_id": "device-001",
    "device_type": "web",
    "os": "Windows",
    "os_version": "11",
    "app_version": "1.0.0"
}
```

**Error Message Scenarios**

1. Missing Required Fields
```json
{
  "message": "email and password is required.",
  "statusCode": 400,
  "messageCode": "error.fieldRequired"
}

```

2. Invalid Email Format
```json
{
  "message": "Invalid email address.",
  "statusCode": 400,
  "messageCode": "error.emailInvalid"
}

```

3. Invalid Credentials (or Admin Not Found)
```json
{
  "message": "Invalid email or password.",
  "statusCode": 400,
  "messageCode": "error.adminLoginInvalid"
}

```

4. Admin Deactivated
```json
{
  "message": "Your account has been deactivated. Please contact support.",
  "statusCode": 400,
  "messageCode": "error.adminDeactivated"
}
```

5. Invalid JSON Body
```json
{
  "message": "Invalid JSON body.",
  "statusCode": 400,
  "messageCode": "error.invalidJson"
}

```
