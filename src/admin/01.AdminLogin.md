# 01. Admin Login

This API is used to authenticate an admin and receive a session token.

**URL:**
`/admin/auth/login`

**Method:**
`POST`

**Headers:**
```
Content-Type: application/json
```

**Sample Input**

```json
{
    "email": "admin@example.com",
    "password": "secure_password",
    "deviceInfo": {
        "deviceId": "sada",
        "deviceType": "android",
        "os": "android",
        "osVersion": "1.2",
        "appVersion": "1.0",
        "notificationToken": "token-123..."
    }
}
```

**Error Message Scenarios**

1. Missing Required Fields
```json
{
    "message": "email and password are required."
}
```

2. Invalid Email Format
```json
{
    "message": "Invalid email address."
}
```

3. Invalid Credentials (or Admin Not Found)
```json
{
    "message": "Invalid email or password."
}
```

4. Admin Deactivated
```json
{
    "message": "Admin account is deactivated."
}
```

5. Invalid JSON Body
```json
{
    "message": "Invalid JSON body."
}
```
