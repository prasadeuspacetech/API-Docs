# 05. Country CRUD

This API is used by Admins to manage countries.

**Base URL:** `/api/auth/admin/country`

---

## 1. List All Countries
Returns a list of all countries.

**URL:** `/api/auth/admin/country`
**Method:** `GET`
**Headers:**
`Authorization: Bearer <Admin_Token>`
`x-app-type: admin`

**Response:**
```json
{
    "countries": [
        {
            "id": "cn001",
            "country_name": "India"
        }
    ],
    "message": "Success"
}
```

---

## 2. Add Country
Adds a new country.

**URL:** `/api/auth/admin/country`
**Method:** `POST`
**Headers:**
`Authorization: Bearer <Admin_Token>`
`x-app-type: admin`

**Request Body:**
```json
{
    "country_name": "Canada"
}
```

**Response:**
```json
{
    "message": "Country added successfully"
}
```

---

## 3. Update Country
Updates an existing country name.

**URL:** `/api/auth/admin/country/<country_id>`
**Method:** `PUT`
**Headers:**
`Authorization: Bearer <Admin_Token>`
`x-app-type: admin`

**Request Body:**
```json
{
    "country_name": "Canada Updated"
}
```

**Response:**
```json
{
    "message": "Country updated successfully"
}
```

---

## 4. Delete Country
Soft deletes a country.

**URL:** `/api/auth/admin/country/<country_id>`
**Method:** `DELETE`
**Headers:** `Authorization: Bearer <Admin_Token>`

**Response:**
```json
{
    "message": "Country deleted successfully"
}
```
