# 02. Religion CRUD

This API is used by Admins to manage religions.

**Base URL:** `/api/auth/admin/religion`

---

## 1. List All Religions
Returns a list of all religions.

**URL:** `/api/auth/admin/religion`
**Method:** `GET`
**Headers:**
`Authorization: Bearer <Admin_Token>`
`x-app-type: admin`

**Response:**
```json
{
    "religions": [
        {
            "id": "r001",
            "religion_name": "Hindu"
        },
        {
            "id": "r002",
            "religion_name": "Muslim"
        }
    ],
    "message": "Success"
}
```

---

## 2. Add Religion
Adds a new religion.

**URL:** `/api/auth/admin/religion`
**Method:** `POST`
**Headers:**
`Authorization: Bearer <Admin_Token>`
`x-app-type: admin`

**Request Body:**
```json
{
    "religion_name": "Sikh"
}
```

**Response:**
```json
{
    "message": "Religion added successfully"
}
```

---

## 3. Update Religion
Updates an existing religion name.

**URL:** `/api/auth/admin/religion/<religion_id>`
**Method:** `PUT`
**Headers:**
`Authorization: Bearer <Admin_Token>`
`x-app-type: admin`

**Request Body:**
```json
{
    "religion_name": "Sikhism"
}
```

**Response:**
```json
{
    "message": "Religion updated successfully"
}
```

---

## 4. Delete Religion
Soft deletes a religion.

**URL:** `/api/auth/admin/religion/<religion_id>`
**Method:** `DELETE`
**Headers:** `Authorization: Bearer <Admin_Token>`

**Response:**
```json
{
    "message": "Religion deleted successfully"
}
```
