# 02. Religion CRUD

This API is used by Admins to manage religions.

**Base URL:** `/api/auth/admin/religion`

---

## 1. List All Religions
Returns a list of all religions.

**URL:** `/api/auth/admin/religion`
**Method:** `GET`
**Headers:**
`Authorization: Bearer <Admin_Token>`
`x-app-type: admin`
`Accept-Language`: `en-US`

## 2. Add Religion
Adds a new religion.

**URL:** `/api/auth/admin/religion`
**Method:** `POST`
**Headers:**
`Authorization: Bearer <Admin_Token>`
`x-app-type: admin`

**Request Body:**
```json
{
    "religion_name": "dummy.hindu1"
}

```

**Response:**
```json
{
    "message": "Religion added successfully.",
    "statusCode": 200,
    "messageCode": "success.religion_added"
}
```

---

## 3. Update Religion
Updates an existing religion name.

**URL:** `/api/auth/admin/religion/<religion_id>`
**Method:** `PUT`
**Headers:**
`Authorization: Bearer <Admin_Token>`
`x-app-type: admin`
`Accept-Language`: `en-US`

**Request Body:**
```json
{
    "religion_name": "dummy.hindu12"
}

```

**Response:**
```json
{
    "message": "Religion updated successfully.",
    "statusCode": 200,
    "messageCode": "success.religion_updated"
}
```

---

## 4. Delete Religion
Soft deletes a religion.

**URL:** `/api/auth/admin/religion/<religion_id>`
**Method:** `DELETE`
**Headers:** 
`Authorization: Bearer <Admin_Token>`
`x-app-type: admin`
`Accept-Language`: `en-US`

**Response:**
```json
{
    "message": "Religion deleted successfully.",
    "statusCode": 200,
    "messageCode": "success.religion_deleted"
}
```
