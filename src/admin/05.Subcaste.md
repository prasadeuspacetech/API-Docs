# 04. Subcaste CRUD

This API is used by Admins to manage subcastes.

**Base URL:** `/api/auth/admin/subcast`

---

## 1. List Subcastes
Returns a list of subcastes. Can be filtered by `caste_id`.

**URL:** `/api/auth/admin/subcast`
**Method:** `GET`
**Headers:**
`Authorization: Bearer <Admin_Token>`
`x-app-type: admin`
**Query Parameters:**
| Parameter | Type | Required | Description |
| :--- | :--- | :--- | :--- |
| `caste_id` | String | No | Filter subcastes by caste |

**Response:**
```json
{
    "subcasts": [
        {
            "id": "sc001",
            "subcaste_name": "Saraswat",
            "caste_id": "c001"
        }
    ],
    "message": "Success"
}
```

---

## 2. Add Subcaste
Adds a new subcaste.

**URL:** `/api/auth/admin/subcast`
**Method:** `POST`
**Headers:**
`Authorization: Bearer <Admin_Token>`
`x-app-type: admin`

**Request Body:**
```json
{
    "subcaste_name": "Gaud",
    "caste_id": "c001"
}
```

**Response:**
```json
{
    "message": "Subcaste added successfully"
}
```

---

## 3. Update Subcaste
Updates an existing subcaste name.

**URL:** `/api/auth/admin/subcast/<subcast_id>`
**Method:** `PUT`
**Headers:** `Authorization: Bearer <Admin_Token>`

**Request Body:**
```json
{
    "subcaste_name": "Gaud Updated"
}
```

**Response:**
```json
{
    "message": "Subcaste updated successfully"
}
```

---

## 4. Delete Subcaste
Soft deletes a subcaste.

**URL:** `/api/auth/admin/subcast/<subcast_id>`
**Method:** `DELETE`
**Headers:** `Authorization: Bearer <Admin_Token>`

**Response:**
```json
{
    "message": "Subcaste deleted successfully"
}
```
