# 04. Subcaste CRUD

This API is used by Admins to manage subcastes.

**Base URL:** `/api/auth/admin/subcast`

---

## 1. List Subcastes
Returns a list of subcastes. Can be filtered by `caste_id`.

**URL:** `/api/auth/admin/subcast`
**Method:** `GET`
**Headers:**
`Authorization: Bearer <Admin_Token>`
`x-app-type: admin`
`Accept-Language`: `en-US`

**Query Parameters:**
| Parameter | Type | Required | Description |
| :--- | :--- | :--- | :--- |
| `caste_id` | String | No | Filter subcastes by caste |

## 2. Add Subcaste
Adds a new subcaste.

**URL:** `/api/auth/admin/subcast`
**Method:** `POST`
**Headers:**
`Authorization: Bearer <Admin_Token>`
`x-app-type: admin`
`Accept-Language`: `en-US`

**Request Body:**
```json
{
    "subcaste_name": "Deshastha123",
    "caste_id": "ca01a1b2c3d4e5f6a1b2c3d4e5f6a1b2"
}
```

**Response:**
```json
{
    "message": "Subcast added successfully.",
    "statusCode": 200,
    "messageCode": "success.subcast_added"
}
```

---

## 3. Update Subcaste
Updates an existing subcaste name.

**URL:** `/api/auth/admin/subcast/<subcast_id>`
**Method:** `PUT`
**Headers:** `Authorization: Bearer <Admin_Token>`

**Request Body:**
```json
{
    "subcaste_name": "Deshastha12"
}
```

**Response:**
```json
{
    "message": "Subcast updated successfully.",
    "statusCode": 200,
    "messageCode": "success.subcast_updated"
}
```

---

## 4. Delete Subcaste
Soft deletes a subcaste.

**URL:** `/api/auth/admin/subcast/<subcast_id>`
**Method:** `DELETE`
**Headers:** `Authorization: Bearer <Admin_Token>`

**Response:**
```json
{
    "message": "Subcast deleted successfully.",
    "statusCode": 200,
    "messageCode": "success.subcast_deleted"
}
```
