# 09. Admin Logout

This API is used by Admins to logout and invalidate their session token.

**URL:** `/api/auth/admin/logout`

**Method:** `POST`

**Headers:**
`Authorization: Bearer <Admin_Token>`
`x-app-type: admin`

**Response:**
*   **Success (200 OK)**:
```json
{
    "message": "Logged out successfully"
}
```

**Error Message Scenarios**

1. Invalid or Missing Token (401 Unauthorized)
```json
{
    "message": "Token is invalid or expired"
}
```

2. Missing `x-app-type: admin` header (403 Forbidden)
```json
{
    "message": "This endpoint requires x-app-type: admin header."
}
```
