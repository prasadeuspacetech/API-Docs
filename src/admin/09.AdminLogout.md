# Admin Logout

This API is used by Admins to logout and invalidate their session token.

**URL:** `/api/auth/admin/logout`

**Method:** `POST`

**Headers:**
`Authorization: Bearer <Admin_Token>`
`x-app-type: admin`
`Accept-Language: en-US`

**Response:**
*   **Success (200 OK)**:
```json
{
    "message": "Logged out successfully.",
    "statusCode": 200,
    "messageCode": "success.logout"
}
```

**Error Message Scenarios**

1. Invalid or Missing Token (401 Unauthorized)
```json
{
  "message": "Authorization token is required.",
  "statusCode": 401,
  "messageCode": "error.tokenRequired"
}

```

2. Missing `x-app-type: admin` header (403 Forbidden)
```json
{
  "message": "This endpoint requires x-app-type: admin header.",
  "statusCode": 403,
  "messageCode": "error.adminAppTypeRequired"
}
```
