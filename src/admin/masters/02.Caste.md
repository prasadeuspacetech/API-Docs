#  Caste CRUD

This API is used by Admins to manage castes.

**Base URL:** `/api/auth/admin/caste`

---

## 1. List Castes
Returns a list of castes. Can be filtered by `religion_id`.

**URL:** `/api/auth/admin/caste`
**Method:** `GET`
**Headers:**
`Authorization: Bearer <Admin_Token>`
`x-app-type: admin`
`Accept-Language`: `en-US`

**Query Parameters:**
| Parameter | Type | Required | Description |
| :--- | :--- | :--- | :--- |
| `religion_id` | String | No | Filter castes by religion |

## 2. Add Caste
Adds a new caste.

**URL:** `/api/auth/admin/caste`
**Method:** `POST`
**Headers:**
`Authorization: Bearer <Admin_Token>`
`x-app-type: admin`
`Accept-Language`: `en-US`

**Request Body:**
```json
{
    "caste_name": "Brahmin12",
    "religion_id": "r001a1b2c3d4e5f6a1b2c3d4e5f6a1b2"
}
```

**Response:**
```json
{
    "message": "Caste added successfully.",
    "statusCode": 200,
    "messageCode": "success.caste_added"
}
```

---

## 3. Update Caste
Updates an existing caste name.

**URL:** `/api/auth/admin/caste/<caste_id>`
**Method:** `PUT`
**Headers:** 
`Authorization: Bearer <Admin_Token>`
`Accept-Language`: `en-US`

**Request Body:**
```json
{
    "caste_name": "Brahmin1234",
    "religion_id": "r001a1b2c3d4e5f6a1b2c3d4e5f6a1b2"
}
```

**Response:**
```json
{
    "message": "Caste updated successfully.",
    "statusCode": 200,
    "messageCode": "success.caste_updated"
}
```

---

## 4. Delete Caste
Soft deletes a caste.

**URL:** `/api/auth/admin/caste/<caste_id>`
**Method:** `DELETE`
**Headers:** 
`Authorization: Bearer <Admin_Token>`
`Accept-Language`: `en-US`

**Response:**
```json
{
    "message": "Caste deleted successfully.",
    "statusCode": 200,
    "messageCode": "success.caste_deleted"
}
```
