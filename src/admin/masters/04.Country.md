# Country CRUD

This API is used by Admins to manage countries.

**Base URL:** `/api/auth/admin/country`

---

## 1. List All Countries
Returns a list of all countries.

**URL:** `/api/auth/admin/country`
**Method:** `GET`
**Headers:**
`Authorization: Bearer <Admin_Token>`
`x-app-type: admin`
`Accept-Language`: `en-US`

**Response:**
```json
[
    {
        "id": "a1b2c3d4e5f6a1b2c3d4e5f6a1b2c3d4",
        "country_name": "India"
    },
    {
        "id": "9ecd7733e4c741ed9dcadc85b2790184",
        "country_name": "test country"
    },
    {
        "id": "c5ba78f02c3346caa6ffd1dd213fb033",
        "country_name": "test country 1"
    }
]
```

---

## 2. Add Country
Adds a new country.

**URL:** `/api/auth/admin/country`
**Method:** `POST`
**Headers:**
`Authorization: Bearer <Admin_Token>`
`x-app-type: admin`

**Request Body:**
```json
{
    "country_name": "Canada"
}
```

**Response:**
```json
{
    "message": "Country added successfully.",
    "statusCode": 200,
    "messageCode": "success.country_added"
}
```

---

## 3. Update Country
Updates an existing country name.

**URL:** `/api/auth/admin/country/<country_id>`
**Method:** `PUT`
**Headers:**
`Authorization: Bearer <Admin_Token>`
`x-app-type: admin`

**Request Body:**
```json
{
    "country_name": "Canada1"
}
```

**Response:**
```json
{
    "message": "Country updated successfully.",
    "statusCode": 200,
    "messageCode": "success.country_updated"
}
```

---

## 4. Delete Country
Soft deletes a country.

**URL:** `/api/auth/admin/country/<country_id>`
**Method:** `DELETE`
**Headers:** `Authorization: Bearer <Admin_Token>`

**Response:**
```json
{
    "message": "Country deleted successfully.",
    "statusCode": 200,
    "messageCode": "success.country_deleted"
}
```
