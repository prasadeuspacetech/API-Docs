# City CRUD

This API is used by Admins to manage cities.

**Base URL:** `/api/auth/admin/city`

---

## 1. List Cities
Returns a list of cities. Can be filtered by `state_id`.

**URL:** `/api/auth/admin/city`
**Method:** `GET`
**Headers:**
`Authorization: Bearer <Admin_Token>`
`x-app-type: admin`
`Accept-Language: en-US`

**Query Parameters:**
| Parameter | Type | Required | Description |
| :--- | :--- | :--- | :--- |
| `state_id` | String | No | Filter cities by state |

## 2. Add City
Adds a new city.

**URL:** `/api/auth/admin/city`
**Method:** `POST`
**Headers:**
`Authorization: Bearer <Admin_Token>`
`x-app-type: admin`
`Accept-Language: en-US`

**Request Body:**
```json
{
    "city_name": "Dummy Pune12",
    "state_id": "s001a1b2c3d4e5f6a1b2c3d4e5f6a1b2"
}
```

**Response:**
```json
{
    "message": "City added successfully.",
    "statusCode": 200,
    "messageCode": "success.city_added"
}
```

---

## 3. Update City
Updates an existing city name.

**URL:** `/api/auth/admin/city/<city_id>`
**Method:** `PUT`
**Headers:** 
`Authorization: Bearer <Admin_Token>`
`Accept-Language: en-US`

**Request Body:**
```json
{
    "city_name": "Pune Updated"
}
```

**Response:**
```json
{
    "message": "City updated successfully.",
    "statusCode": 200,
    "messageCode": "success.city_updated"
}
```

---

## 4. Delete City
Soft deletes a city.

**URL:** `/api/auth/admin/city/<city_id>`
**Method:** `DELETE`
**Headers:** `Authorization: Bearer <Admin_Token>`

**Response:**
```json
{
    "message": "City deleted successfully.",
    "statusCode": 200,
    "messageCode": "success.city_deleted"
}
```
