# 03. Caste CRUD

This API is used by Admins to manage castes.

**Base URL:** `/api/auth/admin/caste`

---

## 1. List Castes
Returns a list of castes. Can be filtered by `religion_id`.

**URL:** `/api/auth/admin/caste`
**Method:** `GET`
**Headers:**
`Authorization: Bearer <Admin_Token>`
`x-app-type: admin`
**Query Parameters:**
| Parameter | Type | Required | Description |
| :--- | :--- | :--- | :--- |
| `religion_id` | String | No | Filter castes by religion |

**Response:**
```json
{
    "castes": [
        {
            "id": "c001",
            "caste_name": "Brahmin",
            "religion_id": "r001"
        }
    ],
    "message": "Success"
}
```

---

## 2. Add Caste
Adds a new caste.

**URL:** `/api/auth/admin/caste`
**Method:** `POST`
**Headers:**
`Authorization: Bearer <Admin_Token>`
`x-app-type: admin`

**Request Body:**
```json
{
    "caste_name": "Rajput",
    "religion_id": "r001"
}
```

**Response:**
```json
{
    "message": "Caste added successfully"
}
```

---

## 3. Update Caste
Updates an existing caste name.

**URL:** `/api/auth/admin/caste/<caste_id>`
**Method:** `PUT`
**Headers:** `Authorization: Bearer <Admin_Token>`

**Request Body:**
```json
{
    "caste_name": "Rajput Updated"
}
```

**Response:**
```json
{
    "message": "Caste updated successfully"
}
```

---

## 4. Delete Caste
Soft deletes a caste.

**URL:** `/api/auth/admin/caste/<caste_id>`
**Method:** `DELETE`
**Headers:** `Authorization: Bearer <Admin_Token>`

**Response:**
```json
{
    "message": "Caste deleted successfully"
}
```
