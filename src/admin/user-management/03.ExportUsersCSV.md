# Export Users CSV API

**Description:**  
This API is used by the admin to export a filtered list of users into a downloadable CSV file. It respects the same query parameters as the User List API to ensure admins only export the data they have currently filtered.

---

### **Endpoint Details**
* **URL:** `/api/auth/admin/users/export`
* **Method:** `GET`
* **Content-Type Expected:** None (This endpoint returns a file, not JSON)

---

### **Headers Required**
* `Authorization`: `Bearer <admin_jwt_token>`
* `x-app-type`: `admin`
* `Accept-Language`: `en` (or the preferred language code)

---

### **Query Parameters**
You can pass the exact same query parameters used in the User List API to filter the CSV export.

| Parameter | Type   | Description |
| :--- | :--- | :--- |
| `search` | String | Search term (matches first name, last name, email, or mobile). |
| `status` | Integer| Filter by account status (`1` for Active, `0` for Inactive). |
| `deleted`| Boolean| Filter by deleted status (`true` for soft-deleted, `false` for active records). |

*Note: Pagination (`page`, `limit`) is ignored for exports. The API will automatically pull all matching records.*

---

### **Response**

#### **On Success (200 OK):**
The server will not return JSON. Instead, it will set HTTP headers to trigger a file download in the browser.

**Response Headers:**
```http
Content-Type: text/csv
Content-Disposition: attachment; filename="users_export_20260526_143000.csv"
```
**Response Body:**
The raw string content of the CSV file.

#### **On Error (JSON):**
If validation or authentication fails, standard JSON errors are returned.
```json
{
  "statusCode": 401,
  "messageCode": "error.unauthorized",
  "message": "Invalid or missing token.",
  "data": null
}
```

---

### **CSV Columns Included**
The generated CSV will contain the following 26 columns logically organized:

1. `Platform ID`
2. `First Name`
3. `Last Name`
4. `Gender`
5. `Age`
6. `Account Status` (Active/Inactive)
7. `Profile Completeness %`
8. `Email Address`
9. `Mobile Number` (Including Country Code)
10. `City`
11. `State`
12. `Country`
13. `Marital Status`
14. `Religion`
15. `Caste`
16. `Subcaste`
17. `Mother Tongue`
18. `Education Level`
19. `Education Field`
20. `Profession Category`
21. `Annual Income`
22. `Height` (e.g., 5'8")
23. `Diet`
24. `Smoke`
25. `Drink`
