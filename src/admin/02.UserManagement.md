# 08. User Management

This API is used by Admins to manage users on the platform.

**Base URL:** `/api/auth/admin/users`

---

## 1. Get All Users
Returns a paginated list of users with basic details.

**URL:** `/api/auth/admin/users`
**Method:** `GET`
**Headers:**
`Authorization: Bearer <Admin_Token>`
`x-app-type: admin`
`Accept-Language`: `en-US`

**Query Parameters:**
| Parameter | Type | Required | Description |
| :--- | :--- | :--- | :--- |
| `page` | Integer | No | Page number (default 1) |
| `limit` | Integer | No | Records per page (default 10, max 100) |
| `search` | String | No | Search by name or mobile |
| `status` | Integer | No | Filter by status (0 = Inactive, 1 = Active) |
| `deleted` | Boolean | No | Filter by deletion status (`true` or `false`) |

## 2. Update User Status
Activates or deactivates a user account.

**URL:** `/api/auth/admin/users/status/<user_id>`
**Method:** `PATCH`
**Headers:**
`Authorization: Bearer <Admin_Token>`
`x-app-type: admin`
`Accept-Language`: `en-US`

**Request Body:**
```json
{
    "status": 0
}
```
*(0 = Deactivate, 1 = Activate)*

**Response:**
```json
{
    "message": "User deactivated successfully.",
    "statusCode": 200,
    "messageCode": "success.user_deactivated"
}
```

---

## 3. Delete User
Soft deletes a user.

**URL:** `/api/auth/admin/users/<user_id>`
**Method:** `DELETE`
**Headers:**
`Authorization: Bearer <Admin_Token>`
`x-app-type: admin`
`Accept-Language`: `en-US`

**Response:**
```json
{
    "message": "User deleted successfully.",
    "statusCode": 200,
    "messageCode": "success.user_deleted"
}
```

---

## 4. Restore User
Restores a soft-deleted user.

**URL:** `/api/auth/admin/users/restore/<user_id>`
**Method:** `PATCH`
**Headers:**
`Authorization: Bearer <Admin_Token>`
`x-app-type: admin`
`Accept-Language`: `en-US`

**Response:**
```json
{
    "message": "User restored successfully.",
    "statusCode": 200,
    "messageCode": "success.user_restored"
}
```
