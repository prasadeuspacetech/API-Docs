# 06. State CRUD

This API is used by Admins to manage states.

**Base URL:** `/api/auth/admin/state`

---

## 1. List States
Returns a list of states. Can be filtered by `country_id`.

**URL:** `/api/auth/admin/state`
**Method:** `GET`
**Headers:**
`Authorization: Bearer <Admin_Token>`
`x-app-type: admin`
`Accept-Language`: `en-US`

**Query Parameters:**
| Parameter | Type | Required | Description |
| :--- | :--- | :--- | :--- |
| `country_id` | String | No | Filter states by country |

## 2. Add State
Adds a new state.

**URL:** `/api/auth/admin/state`
**Method:** `POST`
**Headers:**
`Authorization: Bearer <Admin_Token>`
`x-app-type: admin`
`Accept-Language`: `en-US`

**Request Body:**
```json
{
    "state_name": "Andhra Pradeshhh12",
    "country_id": "a1b2c3d4e5f6a1b2c3d4e5f6a1b2c3d4"
}
```

**Response:**
```json
{
    "message": "State added successfully.",
    "statusCode": 200,
    "messageCode": "success.state_added"
}
```

---

## 3. Update State
Updates an existing state name.

**URL:** `/api/auth/admin/state/<state_id>`
**Method:** `PUT`
**Headers:** 
`Authorization: Bearer <Admin_Token>`
`x-app-type: admin`
`Accept-Language`: `en-US`

**Request Body:**
```json
{
    "state_name": "Goa Updated"
}
```

**Response:**
```json
{
    "message": "State updated successfully.",
    "statusCode": 200,
    "messageCode": "success.state_updated"
}
```

---

## 4. Delete State
Soft deletes a state.

**URL:** `/api/auth/admin/state/<state_id>`
**Method:** `DELETE`
**Headers:** 
`Authorization: Bearer <Admin_Token>`
`x-app-type: admin`
`Accept-Language`: `en-US`

**Response:**
```json
{
    "message": "State deleted successfully.",
    "statusCode": 200,
    "messageCode": "success.state_deleted"
}
```
