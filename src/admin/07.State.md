# 06. State CRUD

This API is used by Admins to manage states.

**Base URL:** `/api/auth/admin/state`

---

## 1. List States
Returns a list of states. Can be filtered by `country_id`.

**URL:** `/api/auth/admin/state`
**Method:** `GET`
**Headers:**
`Authorization: Bearer <Admin_Token>`
`x-app-type: admin`
**Query Parameters:**
| Parameter | Type | Required | Description |
| :--- | :--- | :--- | :--- |
| `country_id` | String | No | Filter states by country |

**Response:**
```json
{
    "states": [
        {
            "id": "st001",
            "state_name": "Maharashtra",
            "country_id": "cn001"
        }
    ],
    "message": "Success"
}
```

---

## 2. Add State
Adds a new state.

**URL:** `/api/auth/admin/state`
**Method:** `POST`
**Headers:**
`Authorization: Bearer <Admin_Token>`
`x-app-type: admin`

**Request Body:**
```json
{
    "state_name": "Goa",
    "country_id": "cn001"
}
```

**Response:**
```json
{
    "message": "State added successfully"
}
```

---

## 3. Update State
Updates an existing state name.

**URL:** `/api/auth/admin/state/<state_id>`
**Method:** `PUT`
**Headers:** `Authorization: Bearer <Admin_Token>`

**Request Body:**
```json
{
    "state_name": "Goa Updated"
}
```

**Response:**
```json
{
    "message": "State updated successfully"
}
```

---

## 4. Delete State
Soft deletes a state.

**URL:** `/api/auth/admin/state/<state_id>`
**Method:** `DELETE`
**Headers:** `Authorization: Bearer <Admin_Token>`

**Response:**
```json
{
    "message": "State deleted successfully"
}
```
